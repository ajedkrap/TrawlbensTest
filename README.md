# TrawlbensTest

This repository is meant to fulfill the test for TRAWLBENS

## Installation

Use the package manager [Node.Js](https://nodejs.org/en/) to install TrawlbensTest.

```bash
git clone https://github.com/ajedkrap/TrawlbensTest.git
yarn install
yarn android
```
